🔊 Ultrasonic Signal Classifier

Machine Learning Project for Defect Detection in Ultrasonic A-Scan Signals

🔗 View on GitHub 📖 Read Documentation

📊 Project Overview

A complete machine learning pipeline for classifying ultrasonic A-scan signals using:

  • 1D Convolutional Neural Networks (PyTorch)
  • Synthetic signal generation with realistic defects
  • Hyperparameter tuning with Optuna
  • Experiment tracking with MLflow
  • Comprehensive evaluation metrics
  • Interactive Jupyter notebook demo

🚀 Quick Start Guide

1. Installation

pip install -r requirements.txt

2. Run the Demo

python run_demo.py

3. Train the Model

python train.py --config config.yaml --model-dir models

4. Evaluate Performance

python evaluate.py --model models/best_model.pt --config config.yaml

5. Hyperparameter Tuning

python hyperparameter_tuning.py --config config.yaml --n-trials 50

6. View MLflow Results

mlflow ui --backend-store-uri file:./logs/mlflow

7. Interactive Notebook

jupyter notebook notebooks/demo.ipynb
